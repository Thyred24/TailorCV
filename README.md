# TailorCV

TailorCV is a lightweight hiring pipeline web app for teams. Track candidates by stage, assign owners, set next steps and due dates, and keep interview feedback structured in one clean workspace.

TailorCV ekipler için hafif ve modern bir aday takip ve işe alım pipeline uygulamasıdır. Adayları aşamalara göre yönetin, sorumlu atayın, next step ve due date belirleyin, görüşme notları ve değerlendirmeleri tek bir workspace içinde düzenleyin.

## Key features

1) Candidate pipeline  
Move candidates through clear stages and always know current status.

2) Ownership and accountability  
Assign an owner per candidate or per step so nothing gets stuck.

3) Next steps and due dates  
Keep follow ups visible and highlight what needs attention.

4) Interviews and scorecards  
Run consistent interviews with structured feedback and aligned decisions.

5) Clean workspace  
A single source of truth for the whole team.

## Tech stack

1) Next.js  
2) React  
3) TypeScript  
4) Chakra UI
